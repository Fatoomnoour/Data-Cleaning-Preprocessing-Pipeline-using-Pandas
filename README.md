# 🧹 Data Cleaning Project – Retail Sales Dataset

## 📌 Overview
This project demonstrates a complete data cleaning process on a retail sales dataset.  
The dataset initially contained inconsistent city names, missing values, duplicate records, and incorrect date formats.

## 🎯 Objective
To transform raw, messy data into a clean and structured dataset ready for analysis.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

## 🔧 Data Cleaning Steps
- Standardized city names (e.g., cairo → Cairo)
- Removed duplicate records
- Handled missing sales values using mean imputation
- Converted date format into standard format (YYYY-MM-DD)
- Ensured correct data types for all columns

## 📂 Files Included
- `Retail store sales dirty for cleaning.ipynb` → Full cleaning process

## 📊 Result
A clean, consistent dataset ready for analysis and reporting.

## 💡 Key Skills Demonstrated
- Data Cleaning
- Data Preprocessing
- Handling Missing Data
- Data Standardization
- Pandas Data Manipulation

2. Run all cells
3. The cleaned dataset will be generated
